# 📊 Sales & Revenue Analysis Dashboard

An interactive dashboard built using **Streamlit** that allows users to analyze sales and revenue data easily. Upload your own dataset (CSV/Excel) and explore insights through charts, KPIs, and filters.

---

## 🚀 Features

* 📁 Upload CSV or Excel files
* 📊 Visualize revenue trends over time
* 🥇 Identify top-performing products
* 🌍 Analyze sales by region
* 👥 Track salesperson performance
* 🔍 Apply filters (Region, Product)
* 📥 Download filtered dataset
* ⚡ Fast and interactive UI

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* NumPy

## ▶️ How to Run the Project

### 1. Clone the Repository

```
git clone https://github.com/your-username/sales-dashboard.git
cd sales-dashboard
```

---

### 2. Create Virtual Environment (Optional but Recommended)

```
python -m venv venv
venv\Scripts\activate
```

---

### 3. Install Required Libraries

```
pip install -r requirements.txt
```

---

### 4. Run the Application

```
streamlit run app.py
```

---

### 5. Open in Browser

```
http://localhost:8501
```

## 📊 Sample Dataset Format

Make sure your dataset contains columns like:

| Column Name | Description             |
| ----------- | ----------------------- |
| Date        | Sales date              |
| Product     | Product name            |
| Region      | Sales region            |
| Salesperson | Salesperson name        |
| Quantity    | Units sold              |
| Revenue     | Total revenue generated |
